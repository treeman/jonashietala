## Combos

```
Top row
                           X + W                #{ | }
C + K       Q                      , + U          Backspace
    K + F     Z
C + K + F    Delete            W + , + U         Ctrl + W (backspace word)

Home row
    T + H        Escape        N + A             :
S + T         Tab                  A + I           Enter
S +     H      ;               N +     I          vim save
S +     H      "               N +     I          '
S + T + H      ;               N + A + I          vim save

        H                      N                 WNAV
    T                              A             CAPSWORD


Bottom row
V + G         Vim vsplit win  L + (             ^
                                  ( + )         $
V + G + P     Vim close win   L + ( + )         Swedish layer


Thumbs
Space                           E               NUMWORD
```
